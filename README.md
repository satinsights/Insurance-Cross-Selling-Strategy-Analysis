# Insurance-Cross-Selling-Strategy-Analysis

# Insurance Lead Conversion Prediction
I worked on a 50,000-row insurance dataset to identify which customers are likely to convert (buy insurance). The project covers data cleaning, EDA (Exploratory Data Analysis), and visual analysis. My goal was to clean the data and analyze which factors influence whether a customer converts or not. Key steps were: handling missing values using smart imputation strategies, EDA across univariate, bivariate, and multivariate levels, and visual analysis using matplotlib and seaborn.

Iused libraries like pandas (data manipulation), numpy (numerical ops), matplotlib & seaborn (visualization), scipy.stats (statistics), re (regex for column name cleaning), ydata_profiling (auto EDA report).

now I have both files completely. INS_Report.ipynb is actually the EDA & Analysis report that comes after the cleaning done in Ins_Project_V4.ipynb. Together they form a complete project.

# The project has two parts.
First, data cleaning (V4 notebook): loading raw Excel data, cleaning column names using regex, handling missing values with 4 different strategies, encoding the target variable.[Ins_Project V4.ipynb](https://github.com/user-attachments/files/27253020/Ins_Project.V4.ipynb)

Second, EDA report notebook: univariate analysis of numeric and categorical columns, bivariate analysis (numeric vs target, categorical vs target, correlation heatmap), multivariate analysis (pairplot, scatterplots, pivot heatmaps), and business recommendations at each stage.[INS_Report.ipynb](https://github.com/user-attachments/files/27252997/INS_Report.ipynb)
